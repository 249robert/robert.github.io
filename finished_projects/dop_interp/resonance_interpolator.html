Resonance Parameter Interpolater

Tuesday, 05/26/2026

The resonance integral I is involved in calculating resonance escape probabilities and group cross sections above thermal energies. To obtain I, you must know another value, J. The parameter J. To obtain J, you need the following equations:

\[ J(\xi,\beta)\equiv\int_{0}^{\infty} \frac{\Psi(\xi,x)}{\beta+\Psi(\xi,x)} dx \]

\[ \Psi(\xi,x)\equiv\frac{\xi}{2}\int_{\frac{-2E}{\Gamma}}^{\infty} \frac{1}{1+y^2} \left[ \exp{\left(-\frac{(\nu-\nu_{r})^2}{2\nu_{th}^2}\right)}- \exp{\left(-\frac{(\nu+\nu_{r})^2}{2\nu_{th}^2}\right)} \right] dy \]

Real evaluations of this integral rely on numerical integration techniques. The upside is that other people have already done the work of evaluating this function over a spread of values for \(\xi\) and \(\beta\). If your problem scenario's parameters for J don't allign with tabulated values, you will need to perform billinear interpolation. Doing this interpolation is tedious—especially if you are not provided a table of values in plaintext.

I've used this problem as an exercise to learn how to make a simple program in JavaScript. DopInterp lets you enter any \(\beta\) or \(\xi\) value in the bounds of the resonance absorbtion table below [1]. If you would like access to the numbers to make your own interpolator, feel free to grab the arrays that they are stored in. They're hardcoded arrays in JavaScript files because I couldn't figure out how to import csvs with JS.

How to Use

  1. Enter your values for \(\xi\) and j. To obtain j, you will need to use \(\beta=2^{j}\cdot10^{-5}\)
  2. Press Calculate

By default, Draw Crosshair is enabled. This will treat the image of the table as a grid and draw intersecting lines that represent where your J request would lie if it was on the table. It can be disabled by selecting the Disable button before calculating.

If the calculation fails, the border around the input parameter box will turn red. The problematic input will be highlighted in red.
